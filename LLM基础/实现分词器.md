# Unicode encoding
一种编码方法，分别有 UTF-8, UTF-16, UTF-32
我们已经知道8个比特组成一个 字节。
UTF-8 占1-4个字节不等
UTF-16占 2-4个字节不等
UTF-32固定占32个字节。
